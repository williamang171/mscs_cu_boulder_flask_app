# Getting Started

1. Create a virtual environment:

```bash
python -m venv venv
```

2. Activate the virtual environment:

```bash
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

3. Install dependencies:

```bash
pip install Flask
```

4. Set environment variables:

```bash
export FLASK_APP=src/app.py
```

5. Run the app

```bash
flask run
```
